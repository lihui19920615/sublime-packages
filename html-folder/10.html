css 学习之旅~

我们以后尽量只写unicode字体，写宋体和微软雅黑。 \5FAE\8F6F\96C5\9ED1 、 \5B8B\4F53

css是没有语义的，但是有很多选项。

css注释 /* css注释 */ 快捷键一样 ctrl + shift + /

font 综合设置字体样式 。 {font: font-style font-weight font-size/line-height font-family;}

必须按顺序书写，不能更换顺序，各个属性以空格隔开。

必须保留font-size和font-family属性, 其余都可以省略。
